# OpenPLC on Linux

  • OpenPLC环境搭建
    • 点击以了解OpenPLC
$ mkdir ICWR && cd ICWR
$ git clone https://github.com/thiagoralves/OpenPLC_v3.git
$ cd OpenPLC_v3
$ ./install.sh linux

  • OpenPLC_Editor on Linux

作用为本地PLC程序调试使用，无需求可跳过

$ git clone https://github.com/thiagoralves/OpenPLC_Editor.git
$ cd OpenPLC_Editor
$ ./install.sh
$ ./openplc_editor.sh
